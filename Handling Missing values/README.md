# 🧩 Handling Missing Values

## 📌 Overview

This folder contains practical notebooks focused on handling missing data in Machine Learning preprocessing workflows using Python, Pandas, and Scikit-learn.

The notebooks demonstrate different techniques used to identify, analyze, and treat missing values in datasets before model training.

## 🧠 Topics Covered

* Missing value analysis
* Understanding null values
* Detecting missing data
* Mean/Median/Mode imputation
* Handling numerical missing values
* Handling categorical missing values
* Data preprocessing workflows
* Feature engineering with missing data
* Practical preprocessing techniques using pandas and sklearn
